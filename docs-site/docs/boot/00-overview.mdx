# Boot Sequence Overview

This section explains how Zoho OS gets from GRUB's 32-bit entry environment to the 64-bit kernel entry in `kmain()`.

The visualizer below is the same walkthrough you already created, now embedded into the docs so each page can focus on a specific phase.

<BootstrapStepFrame step={0} title="Boot sequence visualizer" />

## Source files

- `src/boot/header.asm`
- `src/boot/main.asm`
- `src/boot/long_mode_init.asm`
- `src/kernel/main.c`

## Step map

1. GRUB handoff
2. Multiboot2 header
3. Entry point and stack
4. Enable PAE
5. Load page tables into `CR3`
6. Set `EFER.LME`
7. Enable paging in `CR0`
8. Load the 64-bit GDT and far jump
9. Run 64-bit entry code and call `kmain()`
